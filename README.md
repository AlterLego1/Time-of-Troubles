# Смутное время 

```
Период конца XVI — начала XVII в. вошёл в историю как Смутное время. Характерными явлениями этого периода стали народные волнения и бунты, нестабильность политической власти, связанная с самозванством, вмешательство иностранных государств в дела России и хозяйственный кризис.
```
